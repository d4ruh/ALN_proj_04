# ALN - Projeto 04

Repositório dedicado ao quarto trabalho da disciplina de Álgebra Linear Numéria do curso de graduação em Ciência de Dados e Inteligência Artificial pela FGV - EMAp. O conteúdo do trabalho está dividido em três jupyter notebooks, referentes às questões 1, 2 e 3 do projeto, tratando, respectivamente, de refletores de Householder, da forma de Hessenberg e de exponenciais de matrizes.
